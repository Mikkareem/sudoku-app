# Sudoku App

Sudoku Game in Android using Jetpack Compose. Also we can solve the External Sudoku in this app.

### Technologies
1) Kotlin
2) Jetpack Compose
3) Custom Board Drawing in Jetpack Compose
4) Jetpack Navigation Compose
5) Kotlin Flows/Coroutines
